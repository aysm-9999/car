# 🚗 Car Animation with GSAP

An interactive car animation project built using HTML, CSS, and GSAP.  
The project demonstrates smooth animations and modern frontend animation techniques.

## 🚀 Features
- Smooth GSAP animations
- Scroll-based effects
- CSS transform and scale animations
- Responsive layout using em units

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
- GSAP (GreenSock Animation Platform)

## 📂 Project Structure
- index.html
